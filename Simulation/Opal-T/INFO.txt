Here, I will describe a little bit more into the inner workings of the Opal-T optimization code as well as explain the extension to using a two beam setup. 

The optimization function was used to minimize emittance (emit_x), beam size (rms_x and rms_s), and the number of particles lost (Nfit). Nfit is the total number of starting particles minus the result "alive" particles. The case that was chosen for the single bunch for the input parameters (variables) and the output parameters:
